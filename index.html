📌 세이펜 핀파일 조회기

출판사를 선택하거나 검색어를 입력하세요

txt 파일을 업로드하면 보유 중인 핀파일을 확인할 수 있어요
📋 내 핀파일 목록 txt 만드는 법
Windows
Mac / Linux
  1. 세이펜 SD카드를 PC에 연결합니다 (카드리더기 또는 OTG)
  2. 탐색기에서 SD카드 드라이브 안의 BOOK 폴더로 이동합니다
  3. 주소창에 cmd를 입력하고 Enter를 눌러 명령 프롬프트를 엽니다
  4. 아래 명령어를 입력하고 Enter를 누릅니다:
    dir /b *.pin > pinlist.txt
  5. 생성된 pinlist.txt 파일을 위 버튼으로 업로드합니다

※ BOOK 폴더 경로 예시: D:\BOOK 또는 E:\BOOK

  1. 세이펜 SD카드를 Mac에 연결합니다 (카드리더기 또는 OTG 어댑터)
  2. 터미널을 열고 SD카드의 BOOK 폴더로 이동합니다:
    cd /Volumes/{SD카드 이름}/BOOK
  3. 아래 명령어로 핀파일 목록을 생성합니다:
    ls *.pin > ~/Desktop/pinlist.txt
  4. 바탕화면에 생성된 pinlist.txt 파일을 위 버튼으로 업로드합니다

※ SD카드 이름 확인: 터미널에서 ls /Volumes/ 입력 후 목록에서 확인
